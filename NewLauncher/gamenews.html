April 14th: The Hidden Stars

This update is big, like really. Here are some notable changes.
See all of them in the changelog file provided with the game.

Most notable changes:

  • Added Stars, which you can earn by getting Achievements
  • Added Achievements
  • Changed the UI a bit
  • You can't place blocks on top of yourself anymore
  • New items in Shop
  • Hid a new secret. Can you find it?
  • Fixed Teleporter Doors
  • Added a couple items in the Shop
  • You can buy Stars with Coins, though there will be a limit as to how many you can buy
  • There are 120 stars hidden in the game. Can you find them all?
  • Added Star Doors. They require set amount of stars to unlock. They will not disappear from the player's save data, though
  • New tiles are in the game. You should go look for them. I'll give you a hint: find the button on all input devices
  • And so much more.
