Growalone is a sandbox 2D game for PCs. It was made in XNA with C# with Monogame and the .NET Framework. Feel free to try it!

COMPATIBILITY

The game will *not* work with Windows XP. It requires .NET Framework 4.5, that's why. And also, because MonoGame doesn't support .NET Framework 4.0. The game WILL, however, work with Windows 7 and newer.

Check out the Screenshots page to find out more about Growalone. Or go to the Help and About page!
